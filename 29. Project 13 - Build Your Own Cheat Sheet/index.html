CSS: Display and Positioning

Position Properties

Name Description
static Default position (does not need to be specified)
relative Positions an element relative to its default static position (need to specify using the offset properties)
absolute Other elements will ignore this element and act like it is not present on the page
fixed Fixes an element to a specific position regardless of user scrolling (should be accompanied with the offset properties)
sticky Sticks an element to the specified position as the page is scrolled further (to be used with the offset properties)
z-index Controls how far back/forward an element appears when elements overlap (to be used with integer values)

Display Properties

Name Description
inline Only takes up amount of space necessary to display the content
block Fills the entire width of the page (can be used with width property)
inline-block Combines features of both inline and block elements (can be used with width and height properties)
float Moves an element as far left or as far right as possible (to be used with left and right properties)
clear Specifies how elements behave when bumping into each other (to be used with left, right, both or none properties)
